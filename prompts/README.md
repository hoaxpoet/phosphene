# prompts/

Historical AI-pair-programming session records. Phosphene is built by Matt
(product/design) working with Claude Code; each file here is the kickoff
prompt that scoped one development session or increment. They are kept as
part of the engineering record — the "why" behind many decisions reads best
in the prompt that commissioned the work — and as worked examples of the
session-prompt discipline (see `.claude/skills/session-prompt-author/`).

Nothing in here is needed to build, run, or contribute to Phosphene. The
authoritative docs live in `docs/`; where a prompt and a doc disagree, the
doc wins (prompts are frozen at their session date and are not maintained).

`docs/prompts/` holds the same kind of record for sessions that were filed
alongside their topic docs.
